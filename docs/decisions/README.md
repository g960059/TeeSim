# Architecture Decision Records

ADRs capture long-lived decisions that are hard to reverse or likely to be questioned later.

Format: `ADR-NNNN-slug.md`

Each ADR has: **Date**, **Status** (Proposed / Accepted / Superseded), **Context**, **Decision**, **Consequences**, **Related artifacts**.
