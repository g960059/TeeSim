# Objectives

Outcomes the product should achieve in steady state:

1. **ASE view coverage** — A trainee can find and identify all 28 ASE-recommended adult TEE views in the simulator using probe manipulation.

2. **Educator assignment** — An educator can assign specific view-finding exercises and review trainee performance.

3. **Asset pipeline throughput** — A new patient CT case can be ingested, segmented, and exported as simulator-ready assets (GLB + VTI + JSON) within a few hours using the 3D Slicer / SlicerHeart pipeline.

4. **Rendering performance** — The simulator runs at 30+ FPS on commodity hardware (2020-era MacBook or equivalent) with a full thoracic scene and volume rendering active.

5. **Anatomical accuracy** — All standard ASE anchor views are validated against reference echocardiographic images by a qualified echocardiographer before release.

6. **Open foundation** — The default public case library uses only bundle-safe open-licensed datasets, enabling free redistribution.
